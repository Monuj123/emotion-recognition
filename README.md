# 🎭 Real-Time Facial Emotion Recognition System

A real-time facial emotion recognition system built using Python, OpenCV, and DeepFace.
The application captures live webcam video, detects faces, analyzes emotions, and displays the dominant emotion directly on the video feed.

# 🚀 Features

    📷 Real-time webcam video capture
    
    😀 Face detection using Haar Cascade Classifier
    
    🧠 Emotion analysis using DeepFace (pre-trained deep learning models)
    
    🖼️ Bounding box drawn around detected faces
    
    📝 Displays dominant emotion on live video
    
    ⌨️ Press q to exit safely
    
# 🛠️ Technologies Used

    Python
    
    OpenCV
    
    DeepFace
    
    NumPy
    
    Haar Cascade Classifier
